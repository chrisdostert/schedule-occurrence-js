Javascript library for calculating schedule occurrences

# Supported runtimes

This library is isomorphic & should be consumable from either nodejs or
web browsers.

# Installation

## NPM
```shell
npm install --save schedule-occurrence
```

## Yarn
```shell
yarn add schedule-occurrence
```
